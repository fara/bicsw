Para comenzar a utilizar BIC debe ingresar con su explorar de internet a la página que el consultor encargado le asignó para dicho fin.

Una vez que ingresa a la página proporcionada por el consultor se le mostrará el siguiente inicio de sesión:




Como se muestra en el inicio de sesión de BIC, en este momento ingresa su usuario, contraseña y selecciona el botón de iniciar sesión para entrar al sistema BIC.

Si es la primera vez que ingresa al sistema, el usuario y contraseña le serán proporcionados por el administrador de BIC.
