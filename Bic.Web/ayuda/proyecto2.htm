Agregar nuevo:
Si se desea crear un nuevo proyecto se presiona el botón , el cual podemos visualizar en la figura anterior. Luego de esto se mostrará la siguiente pantalla:


A través de esta pantalla se puede ingresar los campos Nombre, Descripción, Servidor, Esquema, Usuario y Contraseña que corrresponden al nuevo proyecto a crear.
El sistema corroborará si la información sumistrada es correcta. Para saber si se encuentra establecida la conexión con el DataMart se puede presionar el botón . Si se presiona el botón no se creará ningún proyecto y se volverá a la pantalla de gestión de proyecto. Si presiona y las datos son correctos, se creará el proyecto con los datos ingresados.

A continuación se describen cada uno de los campos que se deben llenar para poder crear un proyecto:
Nombre: identificación única con la cual se identificará al DataMart dentro del sistema BIC.
Descripción: campo libre para ingresar información por el usuario del sistema.
Servidor: lugar físico donde está alojada la información del DataMart a crear.
Esquema: nombre del DataMart a crear.
Usuario: nombre del usuario que nos permitirá acceder al DataMart.
Contraseña: contraseña del usuario que nos permitirá acceder al DataMart.

Editar:
Si se desea modificar un proyecto en especial se presiona el "editar" correspondiente al proyecto a editar. Luego de esto se mostrará la siguiente pantalla:


A través de esta pantalla se puede cambiar los campos Nombre, Descripción, Servidor, Esquema, Usuario y Contraseña que ya tenía asignado el proyecto en cuestión. Por ejemplo si ha cambiado el servidor en donde se aloja el DataMart, en el campo Servidor podemos hacer la actualización correspondiente.
El sistema corroborará si la información sumistrada es correcta. Para saber si se encuentra establecida la conexión con el DataMart se puede presionar el botón . Si se presiona el botón no se hará ningún cambio y se volverá a la pantalla de gestión de proyecto. Si presiona y las datos son correctos, se efectuarán las modificaciones ingresadas.
