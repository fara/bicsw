La administración de Atributos se realiza luego de haber iniciado la sesión en BIC y luego de haber seleccionado el proyecto deseado, allí se presenta la pantalla de gestión de proyectos, donde se debe seleccionar la solapa de atributos:



Agregar nuevo:
Si se desea crear un nuevo atributo se presiona el botón , el cual podemos visualizar en la figura anterior. Luego de esto se mostrará la siguiente pantalla:



A través de esta pantalla se pueden ingresar los campos Nombre, Tabla LookUp, Id, Descripciones e Hijo que corrresponden al nuevo atributo a crear.

El sistema corroborará si la información sumistrada es correcta. Si se presiona el botón no se creará el atributo y se volverá a la pantalla de gestión de atributos. Si presiona y las datos son correctos, se creará el atributo con los datos ingresados.

A continuación se describen cada uno de los campos que se deben llenar para poder crear un atributo:
Nombre: identificación única con la cual se identificará al atributo dentro del proyecto.
Tabla LookUp: es la tabla del modelo (la cual se definió en la solapa Tablas) a la cual el atributo a crear hace referencia.
Id: es el identificador de la tabla lookUp con el que se quiere vincular al atributo
Descripción: se seleccionan las descripciones del atributo que se mostrarán en el reporte (pueden ser más de uno).
Hijo: crea una relación con una Tabla (de las posibles, que se presentan en el combo).

Editar:
Si se desea modificar un atributo en especial se presiona el "editar" correspondiente al atributo a editar. Luego de esto se mostrará la siguiente pantalla:



A través de esta pantalla se puede cambiar los campos Nombre, Tabla LookUp, Id, Descripciones e Hijo.
El sistema corroborará si la información sumistrada es correcta.
Si se presiona el botón no se hará ningún cambio y se volverá a la pantalla de gestión de atributos. Si presiona y las datos son correctos, se efectuarán las modificaciones ingresadas.

Borrar:
Si se desea borrar un atributo en especial se presiona el "borrar" correspondiente al atributo a eliminar.
Si el atributo esta siendo usado por algún filtro o reporte, un mensaje hará tal advertencia y no será posible la eliminación.
Si el atributo puede ser eliminado, se desplegará el siguiente mensaje:



Si luego de esta advertencia se presiona el botón quedará elimanado el atributo del sistema.

En cualquier caso anterior si se presiona el botón el atributo quedará en el sistema tal cual como estaba antes de intentar eliminarlo.

Una vez configurados los atributos, podrán ser usados en los filtros y reportes.
