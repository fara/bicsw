Configuración de reporte
Arriba  Abajo  Siguiente

Luego de haber ingresado en un proyecto en particular del sistema BIC, podemos entrar a la solapa de reportes para crear los mismos. La pantalla que se mostrará una vez que se ingresa a los reportes es la siguiente:


Agregar nuevo:
Si se desea crear un nuevo reporte se presiona el botón , el cual podemos visualizar en la figura anterior. Luego de esto se mostrará la siguiente pantalla:



A través de esta pantalla se pueden ingresar los campos Nombre reporte, Atributos, Métricas y Filtros del reporte a crear.
El sistema corroborará si la información sumistrada es correcta. Si se presiona el botón no se creará el reporte y se volverá a la pantalla de gestión de reportes. Si presiona y las datos son correctos, se creará el reporte con los datos ingresados.
Para configurar un reporte se deben seleccionar atributos, meticas y filtros. Las combinacion posibles son:
·Solo atributos.  
·Atributos y filtros.  
·Atributos y métricas.  
·Atributos, filtros y métricas.  
·Métricas y filtros.  

En el caso de que la información seleccionada no sea correcta aparecerá el siguiente mensaje:
"Imposible generar el reporte, no existe combinacion entre los atributos y las metricas seleccionadas"

Editar:
Si se desea modificar un reporte en especial se presiona el "editar" correspondiente al reporte a editar. Luego de esto se mostrará la siguiente pantalla:



A través de esta pantalla se pueden cambiar los campos Nombre reporte, Atributos, y Filtros.
El sistema corroborará si la información sumistrada es correcta de igual forma que lo hace en agregar reporte.
Si se presiona el botón no se hará ningún cambio y se volverá a la pantalla de gestión de reportes. Si presiona y las datos son correctos, se efectuarán las modificaciones ingresadas.

Borrar:
Si se desea borrar un reporte en especial se presiona el "borrar" correspondiente al reporte a eliminar.
Luego se desplegará el siguiente mensaje:



Si luego de esta advertencia se presiona el botón , quedará elimanado el reporte del sistema.

Si se presiona el botón , el reporte quedará en el sistema tal cual como estaba antes de intentar eliminarlo.
