La administración de Filtros se realiza luego de haber iniciado la sesión en BIC y luego de haber seleccionado el proyecto deseado, allí se presenta la pantalla de gestión de proyectos, donde se debe seleccionar la solapa de filtros:



Agregar nuevo:
Si se desea crear un nuevo filtro se presiona el botón , el cual podemos visualizar en la figura anterior. Luego de esto se mostrará la siguiente pantalla:



A través de esta pantalla se puede ingresar los campos Nombre, Atributo, Descripción, Operador y Valor que corrresponden al nuevo filtro a crear.

El sistema corroborará si la información sumistrada es correcta. Si se presiona el botón no se creará el atributo y se volverá a la pantalla de gestión de atributos. Si presiona y las datos son correctos, se creará el filtro con los datos ingresados.

A continuación se describen cada uno de los campos que se deben llenar para poder crear un atributo:
Nombre: identificación única con la cual se identificará al filtro dentro del proyecto.
Atributo: Se selecciona el atributo para el filtro a crear.
Descripción: Se selecciona la descripción del atributo a la que se desea aplicar el filtro.
Operador: Se selecciona el operador para la condición del filtro.
Valor: Se escribe el valor alfanumérico para formar la condición del filtro a crear.

Editar:
Si se desea modificar un filtro en especial se presiona el "editar" correspondiente al filtro a editar. Luego de esto se mostrará la siguiente pantalla:



A través de esta pantalla se puede cambiar los campos Nombre, Atributo, Descripción, Operador y Valor.
El sistema corroborará si la información sumistrada es correcta.
Si se presiona el botón no se hará ningún cambio y se volverá a la pantalla de gestión de filtros. Si presiona y las datos son correctos, se efectuarán las modificaciones ingresadas.

Borrar:
Si se desea borrar un filtro en especial se presiona el "borrar" correspondiente al filtro a eliminar.
A continuación se desplegará el siguiente mensaje:



   

Si luego de esta advertencia se presiona el botón quedará elimanado el filtro del proyecto.

Si se presiona el botón el filtro quedará en el sistema tal cual como estaba antes de intentar eliminarlo.

Una vez configurados los filtros, podrán ser usados en los reportes.
