Luego de haber ingresado en un proyecto en particular del sistema BIC podemos entrar a la solapa de hechos para crear los mismos. La pantalla que se mostrará una vez que se ingresa a los hechos es la siguiente:



Agregar nuevo:
Si se desea crear un nuevo hecho se presiona el botón , el cual podemos visualizar en la figura anterior. Luego de esto se mostrará la siguiente pantalla:



A través de esta pantalla se puede ingresar un nuevo hecho indicando el nombre que se le va a dar al mismo y seleccionando el hecho en "campo".
Si se presiona el botón no se creará el hecho y se volverá a la pantalla de gestión de hechos. Si presiona y están completos y el nombre del hecho no se repite en el proyecto en cuestión, se creará el hecho con los datos ingresados.

Editar:
Si se desea modificar un hecho en especial se presiona el "editar" correspondiente al hecho a editar. Luego de esto se mostrará la siguiente pantalla:



A través de esta pantalla se puede cambiar los campos Nombre y campo.
El sistema corroborará si la información sumistrada es correcta (que no existan alias duplicadas para el proyecto). Si se presiona el botón no se hará ningún cambio y se volverá a la pantalla de gestión de hechos. Si presiona y las datos son correctos, se efectuarán las modificaciones ingresadas.


Borrar:
Si se desea borrar un hecho en especial se presiona el "borrar" correspondiente al hecho a eliminar. Una vez presionado se desplegará el siguiente mensaje:




Si presiona el botón y el hecho no está siendo utilizado por ninguna métrica se eliminará el hecho del proyecto.
En el caso que el hecho esté siendo utilizado por alguna métrica en particular aparecerá un mensaje que dice:
"No se puede eliminar el hecho ya que está siendo utilizado".
En caso de que se presiona el botón el hecho quedará en el sistema tal cual como estaba antes de intentar eliminarlo.
