Instalación del servidor BIC

En cuanto a requerimientos de hardware no hay ninguna restricción, pero se recomienda un equipo P4 o superior con 1gb de RAM. Este equipo deberá tener conectividad con los servidores donde se encuentran los datawarehouse a utilizar.

Internet Information Server (IIS) - Instalación.
Para instalar BIC el servidor deberá correr como sistema operativo Microsoft Windows 2000 o superior (con el Internet Information Server instalado).
·Dirigirse a Panel de control -> Agregar o quitar componentes de Windows -> Application Server  
·Allí seleccionar:  
1.Internet Information Services  
2.ASP .NET  
3.Application Server Console  
4.Enable network COM+ Access.  
 
Creación del directorio virtual
Una vez instalado el IIS se deberá crear el correspondiente directorio virtual que contendrá BIC.
·Dirjirse a Panel de control -> Herramientas administrativas -> Internet Information services manager  
·Allí dirigirse al nodo raíz : "Default Web Site", hacer clic derecho y elegir la opción "Crear Directorio Virtual"  
1.Alias : "bic"  
2.Path : La carpeta "Web" en el directorio donde se ha bajado BIC del svn o distribución en CD (D:\BIC\web por ejemplo)  
 
Asignación de permisos
Se le deberán dar permisos al proceso ASP.NET sobre la carpeta que contiene BIC.
·Dirigirse al directorio Web físico donde se ha creado el directorio virtual (la carpeta path del punto anterior)  
·Allí hacer clic derecho sobre la carpeta, y seleccionar Propiedades -> Seguridad -> Agregar  
·En la ventana que se abre  
1.Cambiar la locacion desde el botón Locations, y elegir el nombre del equipo local en el que se esta trabajando  
2.Agregar en el cuadro de texto el nombre ASPNET y clickear en el botón "Check names". El sistema reconocerá el nombre del servicio colocando el nombre del equipo como prefijo (EJ: HXWS043/ASPNET). Debe prestarse atención en elegir adecuadamente el nombre del equipo en "Locations"

Verificación de la instalación
Para verificar que la instalación fue exitosa acceder a la aplicación a través del browser ingresando la dirección del directorio virtual y se deberá presentar la pantalla de inicio de sesión de usuario.
