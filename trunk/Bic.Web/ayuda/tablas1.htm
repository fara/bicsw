Las tablas son la forma de estructurar los datos en filas o registros y columnas o atributos. Estas son la que contienen la información de nuestro sistema BIC y las que nos van a permitir generar el resto de los elementos del sistema.

El primer paso para empezar a trabajar con BIC es justamente saber cuales son las tablas que necesitaremos del modelo estrella. Una vez que sabemos cuales son estas tablas, las referenciamos al sistema BIC para poder trabajar con los datos que las mismas contienen.

El modelo estrella tiene dos características principales:
·Una o más tablas de dimensiones, cada una con su llave primaria, con uno o más campos descriptivos que se usan para determinar el nivel de atributos de cada dimensión. Cada dimensión se encuentra en una sola tabla.  
·Una FACT TABLE o tabla de hechos que contiene campos numéricos que están configurados con las llaves primarias de las dimensiones.
