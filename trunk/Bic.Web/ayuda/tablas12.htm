Luego de haber ingresado en un proyecto en particular del sistema BIC podemos entrar a la solapa de tablas para crear las mismos. La pantalla que se mostrará una vez que se ingresa a las tablas es la siguiente:



Agregar nuevo:
Si se desea crear una nueva tabla se presiona el botón , el cual podemos visualizar en la figura anterior. Luego de esto se mostrará la siguiente pantalla:


A través de esta pantalla se puede ingresar una nueva tabla indicando el alias, seleccionando el nombre de la misma, y escribiendo el peso que tendrá dentro del proyecto.
Si se presiona el botón no se creará la tabla y se volverá a la pantalla de gestión de tablas. Si presiona y el alias de la tabla no se repite en el proyecto en cuestión, se creará la tabla con los datos ingresados.

Editar:
Si se desea modificar una tabla en especial se presiona el "editar" correspondiente a la tabla a editar. Luego de esto se mostrará la siguiente pantalla:



A través de esta pantalla se puede cambiar los campos Alias y Peso. El campo nombre no es editable.
El sistema corroborará si la información sumistrada es correcta (que no existan alias duplicadas para la tabla del proyecto). Si se presiona el botón no se hará ningún cambio y se volverá a la pantalla de gestión de tablas. Si presiona y el alias no existe en las tablas del proyecto, se efectuarán las modificaciones ingresadas.

Borrar:
Si se desea borrar una tabla en especial se presiona el "borrar" correspondiente de la tabla a eliminar. Una vez presionado se desplegará el siguiente mensaje:



Si presiona el botón y la tabla no está siendo utilizada por ningún atributo o hecho se eliminará la tabla del proyecto.
En el caso que la tabla esté siendo utilizada por algún atributo o hecho en particular, aparecerá un mensaje que dice:
"No se puede eliminar la tabla ya que está siendo utilizado".
En caso de que se presiona el botón la tabla quedará en el sistema tal cual como estaba antes de intentar eliminarla.
